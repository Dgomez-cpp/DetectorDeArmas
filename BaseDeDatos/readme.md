## Base de datos

MongoDB fue utilizado principalmente por su flexibilidad y escalabilidad como base de datos NoSQL. A diferencia de las bases de datos relacionales tradicionales, MongoDB permite almacenar datos en formato de documentos similares a JSON, lo que facilita el manejo de estructuras de datos dinámicas y no estructuradas
